## CYTOOL工具集

通过命令行快速搭建Vue、Gulp预设默认模板的项目。

------
 

### 开发
1. `git clone`

2. `yarn`

3. `npm link`

4. 输入命令:
```yaml
    cytool -h  👉🏼  cytool --help #帮助
    cytool -v  👉🏼  cytool --vue  #创建vue项目
    cytool -g  👉🏼  cytool --glup #创建gulp项目
```
